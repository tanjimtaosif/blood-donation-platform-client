## Project Name:

BloodCare – Blood Donation & Management Platform

## Project Overview:

BloodCare is a modern, life-saving web application built using the MERN stack (React, Node.js, Express, MongoDB) and Firebase. This platform bridges the gap between blood donors and recipients. Users can create blood donation requests, manage donor profiles, and search for donors based on location and blood group. The project features a robust role-based dashboard system for Admins, Volunteers, and Donors, ensuring streamlined operations and secure data management.

## Purpose:

=> BloodCare aims to simplify and accelerate the blood donation process:
=> Facilitate quick search for blood donors during emergencies.
=> Enable users to create, update, and track blood donation requests.
=> Provide a structured platform for volunteers to manage local requests.
=> Collect funds through donations to maintain and scale the platform.
=> Ensure secure and verified user data through modern authentication.

## Key Features:

=> Role-Based Dashboard – Distinct interfaces and functionalities for Admins, Volunteers, and Donors.
=> Secure Authentication – Firebase integration for Email/Password Login and creat account.
=> Advanced Search & Filtering – Find donors or requests by blood group, district, and upazila.
=> Donation Request Management – Full CRUD operations for requests, including status tracking (Pending, In-progress, Done).
=> Admin Analytics – A dedicated panel to view total users, total requests, and total funding statistics.
=> Stripe Payment Integration – A secure gateway for users to contribute funds/donations to the platform.
=> User Management – Admins can manage user roles and update user status (Active/Blocked).
=> Profile Customization – Users can update their personal info, location, and blood group.
=> Responsive Design – Fully optimized for Mobile, Tablet, and Desktop views.
=> Real-time Notifications – Interactive alerts using SweetAlert2 and Toastify for user actions.

## Tech Stack:

=> Frontend: React.js, TailwindCSS, DaisyUI, Framer Motion
=> Backend / Server: Node.js + Express.js (RESTful API)
=> Database: MongoDB (Atlas)
=> Authentication: Firebase Auth & JWT (JSON Web Tokens)
=> Payment Gateway: Stripe API
